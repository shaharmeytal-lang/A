# React Frontend Project - Module 6

## 📌 Project Overview

This project is part of Module 6: Frontend Development.  
The goal is to convert a UI design created in Stitch into a fully working React application.

The application includes multiple pages, reusable components, routing, and a responsive design based on a predefined Design System.

---

## 🚀 Features

- Multi-page React application
- React Router navigation between pages
- Reusable components (Navbar, Footer, Cards, Inputs)
- Responsive design (mobile + desktop)
- Design System based styling (CSS Variables)
- Dummy data only (no backend integration)

---

## 📁 Pages Included

- Home Page (LandingPage)
- Login Page
- Dashboard Page
- Contact Page

---

## 🧩 Components Structure

### Shared Components:
- Navbar
- Footer
- Input Field
- Button
- Card

### Section Components:
- Hero Section
- Featured Cards
- Newsletter Section
- Sidebar

## 🗺️ Routing (Navigation)

| Path | Page |
|------|------|
| / | LandingPage |
| /login | LoginPage |
| /dashboard | DashboardPage |
| /contact | ContactPage |

## 🛠️ Tech Stack

- React
- React Router DOM
- CSS (with Variables)
- JavaScript (ES6+)

---

## 🎨 Design System

The design system is defined in `DESIGN.md` and includes:

- Color palette
- Typography rules
- Spacing system
- Border radius
- Shadows

All styles are implemented using CSS variables in `globals.css`.

---

## 📱 Responsiveness

The application is fully responsive and supports:
- Desktop screens
- Tablets
- Mobile devices

Media queries are used to adapt layouts for smaller screens.
